Lekcija 1: Kako dodati CSS u HTML

Postoje tri načina kako možemo dodati CSS u HTML dokument:

  • Inline CSS – stil pišemo direktno u HTML element
  • Internal CSS – stil pišemo unutar <style> taga u <head>
  • External CSS – stil pišemo u posebnu .css datoteku

Primjeri:

<!-- Inline CSS -->
<p style="color:red;">Ovo je crveni tekst</p>

<!-- Internal CSS -->
<style>
  p { color: blue; }
</style>

<!-- External CSS -->
<link rel="stylesheet" href="style.css">

Važno: Najbolja praksa je koristiti external CSS jer razdvaja sadržaj i dizajn.

Lekcija 2: Opacity i Gradient

U CSS-u možemo mijenjati **prozirnost elemenata** pomoću opacity i stvarati prelaze boja pomoću gradient.

1. Opacity (prozirnost)

Opacity određuje koliko je element proziran. Vrijednosti idu od 0 (potpuno prozirno) do 1 (potpuno neprozirno).

<!-- Primjer opacity -->
<div style="background-color:blue; color:white; padding:20px; opacity:0.5;">
  Ovo je poluprozirni plavi okvir
</div>

2. Gradient (prijelaz boja)

Gradient omogućuje da se boje postepeno mijenjaju, umjesto jednobojnog elementa. Najčešće se koristi linear-gradient.

<!-- Primjer linearnog gradienta -->
<div style="background: linear-gradient(to right, red, yellow); color:black; padding:20px;">
  Ovo je div s prelaskom boja od crvene do žute
</div>

Primjer u slici (opacity + gradient)

Ovako možemo kombinirati oba svojstva:

<div style="background: linear-gradient(to right, purple, pink); padding:20px; color:white; opacity:0.7;">
  Ovaj okvir ima gradient i djelomičnu prozirnost
</div>

Primjer: Kombiniranje Gradienta i Opacity

U ovom primjeru koristimo linearni gradient i opacity zajedno:

<div style="background: linear-gradient(to right, purple, pink); padding:20px; color:white; opacity:0.7;">
  Ovaj okvir ima gradient i djelomičnu prozirnost</div>
Ovaj okvir ima gradient i djelomičnu prozirnost

✅ Napomena: opacity utječe i na tekst unutar elementa, dok background-color ili background-image može biti proziran bez da tekst postane proziran koristeći rgba().

Lekcija 3 - CSS Positioning

CSS Positioning – Najvažnije stvari

1. Što je positioning?

Positioning u CSS-u određuje na koji način se element postavlja unutar web stranice. Postoji 5 osnovnih tipova positioninga.

Vrste positioninga

  • static
  • relative
  • absolute
  • fixed
  • sticky

Z-Index

  • Određuje redoslijed slojeva elemenata (tko je "ispred", tko "iza").
  • Samo elementi s position vrijednošću različitom od static koriste z-index.

Primjer: Z-indexa


<style>
.container {
  position: relative;
}

.black-box {
  position: relative;
  z-index: 1;
  border: 2px solid black;
  height: 100px;
  margin: 30px;
}

.gray-box {
  position: absolute;
  z-index: 3; <!-- gray box will be above both green and black box -->
  background: lightgray;
  height: 60px;  
  width: 70%;
  left: 50px;
  top: 50px;
}

.green-box {
  position: absolute;
  z-index: 2; <!-- green box will be above black box -->
  background: lightgreen;
  width: 35%;
  left: 270px;
  top: -15px;
  height: 100px;
}
</style>


<div class="container">
  <div class="black-box">Black box (z-index: 1)</div>
  <div class="gray-box">Gray box (z-index: 3)</div>
  <div class="green-box">Green box (z-index: 2)</div>
</div>

Lekcija 4: Sjene i zaobljeni rubovi

CSS Box Shadow – Sjene elemenata

Svojstvo box-shadow u CSS-u koristi se za dodavanje sjene iza elemenata (npr. okvira, slika, gumbova, kutija…). Omogućuje da elementi izgledaju istaknutije i modernije.

Osnovni kod:




box-shadow: horizontal vertical blur spread color;

Značenje

  • horizontal – koliko sjena ide udesno (npr. 5px)
  • vertical – koliko ide prema dolje (npr. 5px)
  • blur – koliko je sjena zamućena (npr. 10px)
  • spread – koliko je velika sjena (opcionalno)
  • color – boja sjene (npr. gray ili rgba(0,0,0,0.3))

Primjer(css)




div {
  width: 200px;
  height: 100px;
  background-color: white;
  box-shadow: 5px 5px 10px gray;
}

Ovaj kod stvara sivu sjenu 5px desno i 5px dolje, s blagim zamućenjem

Primjer unutarnje sjene-inset(css)




div {
  box-shadow: inset 3px 3px 5px gray;
}

Prozirne sjene(css)

Pomoću RGBA boja možeš dobiti nježnije sjene:




div {
  box-shadow: 4px 4px 10px rgba(0, 0, 0, 0.3);
}

Primjena na slike i gumbove(css)

Sjenu možeš koristiti da slike ili gumbi izgledaju profesionalnije.




img {
  box-shadow: 0 0 15px rgba(0,0,0,0.4);
  border-radius: 10px;
}

button {
  box-shadow: 2px 2px 6px gray;
}

Zaobljeni rubovi

Svojstvo border-radius omogućava zaobljene kutove elemenata poput div-ova, slika, gumbova…




selector {
  border-radius: vrijednost;
}

Vrijednosti mogu bit:

  • px (pikseli) npr. 10px
  • % npr. 50% → za krug
  • više vrijednosti → različiti kutovi različito zaobljeni

Lekcija 5: Flexbox

Što je Flexbox

Flexbox (Flexible Box Layout) je CSS način rasporeda elemenata koji omogućava lako poravnanje i prilagodbu unutar kontejnera. Koristi se za izradu fleksibilnih i responzivnih rasporeda bez floatova ili pozicioniranja.

Kada koristiti Flexbox

Koristi se kada želiš:

  • Lako centrirati elemente
  • Jednako rasporediti prostor
  • Mijenjati redoslijed elemenata
  • Napraviti responzivan dizajn (npr. meni, galeriju, raspored blokova)

Osnovni pojmovi

  • Flex container – element koji sadrži flex iteme (display: flex)
  • Flex items – elementi unutar kontejnera
  • Glavna os (main axis) i poprečna os (cross axis)
  • Flex direction – smjer rasporeda elemenata

Postavljanje flexboxa

  • display: flex – blokovni flex container
  • display: inline-flex – inline flex container

Glavne osobine containera

  • flex-direction – redoslijed (row, column, row-reverse, column-reverse)
  • flex-wrap – lomljenje elemenata (nowrap, wrap, wrap-reverse)
  • justify-content – raspored duž glavne osi
  • align-items – poravnanje po poprečnoj osi
  • align-content – razmak između redova

Flex box primjer

Fleksibilni raspored mora imati roditeljski element kojem je svojstvo display postavljeno na flex.

Dimenzije u CSS

Imamo dvije vrste dimenzija, a to su:

  • Absolutne
  • Relativne

    • Absolutne dimenzije

    Glavna mjerna jedinica za absolutne dimenzije je PIXEL(px)

    Također imamo mjerne jedinice kao što su: cm,mm...

    Relativne dimenzije

    Relativne dimenzije zavise od drugih elemenata ili postavki preglednika i omogućuju fleksibilan i responzivan dizajn. Najčešće jedinice

  • % – postotak veličine roditeljskog elementa (npr. width: 50%;)
  • em – relativno u odnosu na veličinu fonta trenutnog elementa (npr. font-size: 2em;)
  • rem – relativno u odnosu na veličinu fonta root elementa (html)
  • vw i vh – postotak širine i visine viewporta (npr. width: 50vw;)
